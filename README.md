# jsp_BBS

- 웹 사이트 개발에서 가장 핵심이 되는 게시판 기능을 JSP를 이용하여 개발 
- 웹 디자인 프레임워크로는 부트스트랩을 이용하며 게시판 만들기 (시판의 글 작성, 글 삭제, 글 보기, 글 목록 등을 구현)
